# Processo Seletivo Maxiprod
### Tecnologias usadas
**Backend:**
- ASP.NET Core 10.0;
- Entity Framework Core;


### Rodar o sistema
**Backend:**

Abra um terminal na pasta *webapi*. Em seguida, execute os seguintes comandos:

*Rodar as migrations:*
```bash
dotnet ef database update
```

*Executar o backend:*
```bash
dotnet run
```
